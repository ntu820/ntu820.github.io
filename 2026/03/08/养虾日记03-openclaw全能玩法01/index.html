养虾日记03-openclaw全能玩法：✨远程 + 换模型 + 浏览器插件

从只能蹲本地电脑用，到随时随地远程撸、换上超顶的大模型、还能嵌到浏览器里，全程保姆级教程，新手也能秒会💪

一：🖥️ 告别本地限制！远程访问 OpenClaw

默认的 127.0.0.1 真的拴 Q，搭在服务器上根本摸不着！

两种方法直接拿捏👇

1. ✅ 懒人 LAN 模式（简单但注意安全）

按照默认配置的openclaw，默认只能在当前安装的电脑上进行操作，且使用地址只能是127.0.0.1，这样对于搭建在服务器侧的龙虾，玩虾体验感很不好，这里介绍几种可以方法。

  1. 直接在安装的实时，直接绑定到LAN地址，不要使用loopback。

    或者手动编辑.openclaw/openclaw.json配置文件

2. ✅ 安全党必冲 SSH 隧道（重点！）

  1. 直接开放LAN容易引起一些安全风险，毕竟龙虾还是在本地”养养“就好了。这里使用SSH隧道技术，首先保证在龙虾的电脑上安装好了SSH服务。

  2. 📋 配置 SSH 访问，如果已经有可以远程的账号，下面可以省略

  3. 📋 在你本地电脑连接

    这条命令的本质是：在你的本地电脑和远程服务器（OpenClaw 所在的机器）之间，建立一条加密的 “专属通道”，把远程服务器上 18789 端口的服务（比如 OpenClaw），映射到你本地电脑的 18789 端口上。

    简单说：你在本地访问 127.0.0.1:18789，就等于直接访问远程服务器的 18789 端口，实现了 “远程访问 OpenClaw” 的核心需求。

    当然如果你有可视化的代理ssh工具，也可以直接在工具中配置，这里以MobaXterm为例

  4. 到此即可在其他电脑上访问服务器侧的龙虾

二：🧠 大脑升级！一键换超牛模型

OpenClaw 创始人 Peter Steinberger 公布了一份 32 个模型的真实测试排名。评价维度只有三个：成功率/速度/费用，但都非常关键。简单说就是：能不能做成事、做得多快、花多少钱。

成功率排名，前五名是：

1️⃣ google/gemini-3-flash-preview

2️⃣ minimax/minimax-m2.1

3️⃣ moonshotai/kimi-k2.5

4️⃣ anthropic/claude-sonnet-4.5

5️⃣ google/gemini-3-pro-preview

速度排名，前五名是：

1️⃣ minimax/minimax-m2.5

2️⃣ google/gemini-2.0-flash

3️⃣ meta-llama/llama-3.1-70b

4️⃣ google/gemini-1.5-pro

5️⃣ mistral/mistral-large

费用排名，最便宜的几个模型是：

1️⃣ GPT-5 Nano

2️⃣ gemini-2.5-flash-lite

3️⃣ mistralai/devstral-2512

4️⃣ GPT-4o-mini

5️⃣ minimax-m2.1

OpenClaw 官方实测 32 个模型，我扒了最优解👇

三：🌐 嵌进浏览器！插件搭建超简单

1. 对比：openclaw对比chrome

  • openclaw：托管的、独立的浏览器（无需扩展程序）。
  • chrome：将扩展程序转发到您的系统浏览器（需要将 OpenClaw 扩展程序附加到选项卡）。

设置browser.defaultProfile: "openclaw"是否默认启用管理模式。

这里设置成chrmoe，使用谷歌浏览器扩展方式。

终于能看到 OpenClaw 咋操作了！放弃默认的 openclaw 模式，冲 Chrome 插件版👇

2. 安装谷歌浏览器插件

  1. 将扩展程序安装到本地

  2. 找到已安装扩展程序的目录路径，一般在 ~/.openclaw/brower/...

  3. 打开谷歌浏览器 -> chrome://extensions

    • 启用”开发者模式“
    • “加载已解压的目录”→ 选择上方显示的目录
    • 固定扩展程序
