👇小白也能会的切换教程

重要说明：从 2022 年 1 月 1 日起，Anaconda 的 defaults 渠道对商业用户收费，虽然个人和学术用途仍可免费使用，但为了规避潜在的许可风险，推荐使用完全开源免费的 conda-forge 渠道。

第一步：下载 Miniforge

注：以下下载方式选一个就行，推荐清华大学开源软件镜像站

  • 方式一（推荐）：清华大学开源软件镜像站
  • 方式二：GitHub Releases（可能较慢）
  • 方式三：南京大学开源镜像站

找到并下载： Miniforge3-Windows-x86_64.exe

安装前准备

  • 关闭所有正在运行的命令行窗口（CMD、PowerShell）
  • 临时禁用杀毒软件（某些安全软件会拦截安装过程）

第二步：安装（重点看！）

  1. 运行安装程序：双击下载的 .exe 文件。

  2. 点击 “Next” → “I Agree”

  3. 选择 “Just Me” → “Next”

  4. 安装位置填写：D:\soft\miniforge3 → “Next”

  5. 勾选以下 3 个选项 → “Install”：

    • ✅ Create shortcuts (supported packages only)
    • ✅ Add Miniforge3 to my PATH environment variable
    • ✅ Register Miniforge3 as my default Python 3.12

  6. 完成后点击 “Next” → “Finish”

第三步：配置国内源（关键！）

在用户目录 C:\Users\<你的用户名>\ 下新建文本文档，命名为 .condarc。（别漏小数点❗）

小贴士：

  • 如果不知道用户名：桌面右键 → “在终端中打开” → 输入 whoami → 回车，\ 后面的就是用户名（如 hsxbk\hsx，则 hsx 是用户名）
  • 新建文本文档默认扩展名为 .txt，需手动删除 .txt
  • 若看不到扩展名：文件资源管理器 → “查看” → “选项” → 取消勾选 “隐藏已知文件类型的扩展名”
