maven制作archetype

简介

Archetype是Maven工程的模板工具包。使用Archetype提供的方法可以快速的构建和组织一致化的工程，解决在工作过程中创建一个新项目时需一堆繁琐的配置工作等。

基于maven项目制作archetype

  1. 准备好maven项目，将无关代码全部删除，只保留最基础的项目和结构

  2. 在项目的根目录下的pom中添加依赖

  3. 进入项目根目录，执行mvn archetype:create-from-project进行模板生成

  4. 进入项目的**/target/generated-sources/archetype**目录，执行mvn clean install

  5. idea使用自定义archetype

    1. 建立自己的项目模板

      1.==archetype-catalog.xml==一般在==.m2==的目录下，默认为用户家目录

      2.如果修改过maven的仓库保存路径的，一般就在修改过的目录下，可以找到==archetype-catalog.xml==文件

      3.为了正确执行生成操作，建议在用户家目录下新建==.m2== 文件夹，然后把maven的==settings.xml==和==archetype-catlog.xml==放进去

    2. 填写自己项目的信息，即可进行生成

  6. 删除自定义的archetype

    一般在==C:\Users\Administrator\AppData\Local\JetBrains\IntelliJIdea2020.1\Maven\Indices==或者==C:\Users\Administrator.idea\Maven\Indices==文件夹下，找到==UserArchetypes.xml==文件，去除想要删除archetype模板，重启idea即可
