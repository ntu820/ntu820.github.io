NFS服务搭建

Centos7安装及配置 NFS 服务器

步骤 1：安装 NFS 服务器 (NFS kernel server)

若要设置服务器机器，就必须在机器上安装 NFS 服务器。

  1. 安装 NFS 服务器。
    yum -y install nfs-utils rpcbind

步骤 2：创建输出目录

NFS 客户端将在服务器机器上挂载一个目录，该目录已由 NFS 服务器输出。

  1. 运行以下命令来指定挂载文件夹名称
    mkdir -p /home/zjnfs
chmod -R a+w /home/zjnfs

步骤 3：授予客户端机器访问 NFS 服务器的权限

  1. 运行以下命令：

    vi /etc/exports

  2. 将客户端信息添加到文件中。

    /home/zjnfs *(rw,sync,no_root_squash)

步骤4：启动服务

  1. 启动

    systemctl start rpcbind
systemctl start nfs

步骤 4：应用配置

  1. 运行以下命令输出共享目录。

    exportfs -arv

使用showmount命令测试NFS输出目录状态

showmount -e
# 客户端测试
showmount -e [NFS服务器IP]
