2.使用Middlewares

1.添加基础认证，BasicAuth

具体参考：https://doc.traefik.io/traefik/middlewares/http/basicauth

  1. 为基础认证设置账密

    apiVersion: v1
kind: Secret
type: Opaque
metadata:
  name: demo-secret
  namespace: default
data:
  # 加密串可以由： htpasswd -nb user password | openssl base64 获得
  demo1-auth: dXNlcjokYXByMSRxUGlReWlTaiQxY3RpYXczWGFzWFIyb1RodmY1T08vCgo=

  2. 配置middleware

    apiVersion: traefik.containo.us/v1alpha1
kind: Middleware
metadata:
  name: demo-auth-middle
  namespace: default
spec:
  basicAuth:
    secret: demo-secret # 使用刚刚生成的密钥
    removeHeader: true # 流量离开traefik时，移除认证，不要和后端自己的认证冲突
    headerField: X-WebAuth-User # 指定一个变量来存储用户名称

  3. 修改路由

    apiVersion: traefik.containo.us/v1alpha1
kind: IngressRoute
metadata:
  name: demo-router
  namespace: default
spec:
  entryPoints:
    - web
  routes:
  - kind: Rule
    match: Host(`demo.app.cn`)
    middlewares:
    - name: demo-auth-middle # 增加认证
      namespace: default
    services:
    - name: demo-svc
      namespace: default
      port: 8080

  4. 验证

2.截取url

  1. 配置middleware

    apiVersion: traefik.containo.us/v1alpha1
kind: Middleware
metadata:
  name: strip-path
  namespace: default
spec:
  stripPrefix:
    prefixes:
    - /demo1

  2. 修改路由

    apiVersion: traefik.containo.us/v1alpha1
kind: IngressRoute
metadata:
  name: demo-router
  namespace: default
spec:
  entryPoints:
    - web
  routes:
  - kind: Rule
    match: Host(`demo.app.cn`) && PathPrefix(`/demo1`)
    middlewares:
    - name: strip-path
      namespace: default
    services:
    - name: demo-svc
      namespace: default
      port: 8080

  3. 验证
